# 修改图标

社交图标全部使用的是阿里图标库，我选了一些最常用的图标，你可以自己挑选

地址为iconfont Demo (cco.vin) (opens new window)


module.exports = {

    themeConfig: {
        socials: [
            {
                aHref: "javascript:;",
                showImgSrc: "https://ooszy.cco.vin/img/blog-public/wechat.jpg",
                isHome: true,
                show: true,
                symbol: '#icon-weixin',
                sidebar: true
            },
        ]

    }
}
